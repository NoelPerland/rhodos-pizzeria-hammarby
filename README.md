# Rhodos Pizzeria Hammarbyhöjden prototype

Unofficial one-page website prototype for Pizzeria Rhodos in Hammarbyhöjden/Johanneshov.

This is not the restaurant's official website. The page keeps a visible prototype disclaimer and should only be used as a preview or sales mockup until the restaurant approves the content.

## Preview locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Deployment

The repository deploys to GitHub Pages through GitHub Actions.

Workflow:

`/.github/workflows/pages.yml`

The deployment runs on pushes to `main` and can also be started manually from the Actions tab.

Expected live URL:

`https://noelperland.github.io/rhodos-pizzeria-hammarby/`
